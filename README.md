# 🚗 Patente B — English Study Lab

**Live app → [ejazfahil.github.io/patenteB_en_feil](https://ejazfahil.github.io/patenteB_en_feil)**

An unofficial study tool for the Italian **Patente B** (driving licence) theory exam, designed for **English speakers with A1 Italian**.

---

## What this is

The Italian Patente B exam is 30 true/false statements in Italian, 20 minutes, and you must score ≤ 3 errors to pass. About 37% of candidates fail — almost always on tricky wording.

This app solves the core tension:

- **The real exam is Italian-only** → the app always shows the Italian statement
- **You think in English (C2)** → every statement has one-tap English translation + explanation

---

## How to study (fast-track guide)

1. **Week 1** — Hit the Glossary first. Learn the 30–40 recurring Italian terms (`segnale`, `divieto`, `obbligo`, `sorpasso`, `sosta`, `corsia`…). These unlock most questions.
2. **Week 2** — Work through Topic Practice with **English help ON**. Read the Italian, then check the English explanation.
3. **Week 3** — Switch to **Exam mode** (English help OFF). Run full simulations. Target < 2 errors per mock.
4. **Ongoing** — Use **Smart Revision** daily for 10 minutes. It automatically resurfaces your weakest questions.

---

## Exam rules (always verify with Motorizzazione)

- 30 statements — mark each VERO or FALSO
- 20 minutes total
- **≤ 3 errors = PASS**

## Tech

Single self-contained HTML file — no server, no install, works offline after first load. All ~7,100 official Ministry questions are embedded with their road-sign images. Progress is saved to `localStorage`.
